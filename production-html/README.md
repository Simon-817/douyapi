# 豆芽Pi v1.0 MVP 生产级 HTML

这是基于 PRD、UI 设计规范、页面切图和透明 PNG 生产素材实现的前端原型。

## 文件

- `mini-program.html`：小程序前端生产 HTML，用于手机浏览器打开，不包含手机外壳样式。
- `admin.html`：运营后台生产 HTML，用于桌面浏览器打开。
- `index.html`：两个独立入口的导航页。
- `styles.css`：品牌样式、响应式布局、小程序和后台界面。
- `mini-program.js`：小程序本地交互、生成流程、兑换码、异常演示。
- `admin.js`：后台筛选用户与增加生成次数交互。
- `assets/`：从开发交付包复制的透明 PNG 生产素材。
- `tests/verify-production-html.js`：结构和素材引用检查脚本。

## 本地运行

```bash
python3 -m http.server 4173
```

打开：

- 小程序前端：`http://127.0.0.1:4173/mini-program.html`
- 运营后台：`http://127.0.0.1:4173/admin.html`

## 交互说明

- 点击“立即体验”进入生成页。
- 点击上传区域模拟上传图片。
- 选择 52、78、104 或自定义 8-104。
- 兑换码 `DOUYA2026` 或 `PI2026` 可增加 3 次生成次数。
- 生成成功后才扣减次数。
- 异常页覆盖上传失败、图片不清晰、内容风险、次数不足、生成超时、保存失败。
- 运营后台支持用户筛选和增加生成次数。
